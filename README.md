# MRI System (Mobile-only)

This repo is the **mobile-first MRI market interpretation PWA**.
No local backend / no PC runner: the app reads `public/data/latest.json`, which is updated by GitHub Actions.

## Run locally

```bash
npm install
npm run dev
```

## Data update

- `public/data/latest.json` is generated by `scripts/update_latest.py`
- GitHub Actions runs every 15 minutes, but only writes:
  - on the normal cadence: **:00 / :30** (with **10-minute delay policy**), or
  - inside **event windows** defined in `public/data/calendar.json` (±15m)

## Deploy (GitHub Pages)

Build and deploy with your preferred GitHub Pages workflow.
Recommended Vite base is already set to `./` for Pages compatibility.

If you use the official Pages deploy action:
- build: `npm ci && npm run build`
- publish: `dist/`


## GitHub Pages (Project Site)

This repo is designed for GitHub Pages project deployment at `/regime-lookout/`.
If you rename the repo, update `vite.config.js -> base` accordingly.
